# BeatCode

> 2023年上海交通大学软件学院暑期课程《互联网产品设计与开发》课程项目。

感谢沈备军老师和任锐老师的授课。

## 项目简介

<img src="./logo.jpg" alt="logo" style="zoom: 20%;" align="center"/>

本项目为一个在线代码评测平台（oj）。
- 前端使用`react`框架
- 后端使用`spring cloud`框架
- 数据库使用`mysql`和`mongodb`
- 缓存使用`spring cache`
- 后端评测结果使用`websocket`实时推送
